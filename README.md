This is a modified version of the ABC Quantum Reactive Scattering Program forked from humberto-jr/abc  originally developed by the group of D. E. Manolopoulos [1].
# ABC

## build and run
```bash 
make
cd ./run 
./a.out < input.d
``` 
or 

```bash
./Build_by_FPM.sh
cd ./run
./ABC < input.d
```
